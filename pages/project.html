The Project

How StudyUp was built

StudyUp is a Web Design course project built entirely from scratch using fundamental web technologies — no frameworks, no shortcuts. Here's an inside look at the stack, structure and development journey.

Tech Stack

Technologies Used

HTML5
Semantic markup, accessible structure and multi-page layout
CSS3
Custom variables, flexbox, grid, media queries and animations
JavaScript
DOM manipulation, event handling, session storage and dynamic UI
PHP
Server-side login, registration and session management
MySQL
User data stored and retrieved via a relational database
Google Fonts
Syne and DM Sans for a modern, clean typography system
Course Requirements

What the project covers

  • Multiple HTML pages with consistent navigation
  • Fully responsive design with media queries
  • Hamburger menu for mobile viewports
  • Meta viewport tag on every page
  • CSS custom properties (variables) for theming
  • JavaScript for dynamic UI interactions
  • User registration & login with PHP + MySQL
  • Client-side form validation
  • Scroll animations and reveal effects
  • Accessible HTML with ARIA labels
  • Footer and consistent page structure across all pages
  • Session-based auth state (show/hide login button)
Development Timeline

How it came together

Week 1
Planning & Wireframing
Defined the project concept, sitemap and page layout. Sketched wireframes for main pages.
Week 2
HTML Structure & CSS Foundation
Built semantic HTML for all pages, set up CSS variables, navbar, footer and base typography.
Week 3
Responsive Design & JavaScript
Added media queries, hamburger menu, scroll animations and dynamic JS interactions.
Week 4
PHP Backend & Final Polish
Integrated PHP login/register, connected MySQL database, tested on all screen sizes and submitted.
